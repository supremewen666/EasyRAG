# Notebooks

EasyRAG 的 notebook 分成两组：

- `fundamentals/`：主线教学，帮助读者先跑通完整流程
- `experiments/`：原理实验，用于对比不同策略和观察现象

## 主线 notebooks / Fundamentals

- [00_quickstart.ipynb](fundamentals/00_quickstart.ipynb): 最小上手闭环
- [01_build_index.ipynb](fundamentals/01_build_index.ipynb): 索引构建与中间产物
- [02_query_modes.ipynb](fundamentals/02_query_modes.ipynb): 查询模式对比
- [03_custom_documents.ipynb](fundamentals/03_custom_documents.ipynb): 自定义文档接入
- [04_knowledge_graph.ipynb](fundamentals/04_knowledge_graph.ipynb): 知识图谱增强检索
- [05_generation_basics.ipynb](fundamentals/05_generation_basics.ipynb): generation 基础路径

## 实验 notebooks / Experiments

- [10_query_preprocessing_lab.ipynb](experiments/10_query_preprocessing_lab.ipynb): query 预处理实验
- [11_chunking_strategy_lab.ipynb](experiments/11_chunking_strategy_lab.ipynb): chunk 策略实验
- [12_vector_retrieval_lab.ipynb](experiments/12_vector_retrieval_lab.ipynb): 向量检索实验
- [13_hybrid_fusion_lab.ipynb](experiments/13_hybrid_fusion_lab.ipynb): 混合检索与融合实验
- [14_rerank_lab.ipynb](experiments/14_rerank_lab.ipynb): rerank 实验
- [15_storage_backend_lab.ipynb](experiments/15_storage_backend_lab.ipynb): backend 对比实验
- [16_failure_cases_and_debugging.ipynb](experiments/16_failure_cases_and_debugging.ipynb): 失败案例与调试实验
- [17_top_k_selection_lab.ipynb](experiments/17_top_k_selection_lab.ipynb): top-k 选择实验
- [18_context_packing_lab.ipynb](experiments/18_context_packing_lab.ipynb): 上下文组装实验
- [19_prompting_and_answer_style_lab.ipynb](experiments/19_prompting_and_answer_style_lab.ipynb): prompt 与回答风格实验
- [20_generation_failure_cases.ipynb](experiments/20_generation_failure_cases.ipynb): generation 失败案例实验
